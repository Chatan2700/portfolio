# Portfolio + DevLog

A minimal personal portfolio and DevLog built with Next.js, TypeScript, Tailwind CSS, and file-based Markdown content.

The goal of this project is to create a simple, fast, recruiter-facing website where I can showcase my work, document technical decisions, and track my growth as a Software Developer focused on full-stack applications and cloud infrastructure.

## Live Site

[sebastianrios.vercel.app](https://sebastianrios.vercel.app)

## Features

- Minimal dark portfolio layout
- About page with career focus and technical stack
- Project archive structure
- Project detail pages
- Markdown-based DevLog
- Dynamic DevLog post routes
- Static content model
- Responsive layout
- Deployed with Vercel

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Content:** Markdown / MDX-style files with frontmatter
- **Deployment:** Vercel
